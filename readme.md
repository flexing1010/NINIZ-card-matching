### <a href="https://flexing1010.github.io/NINIZ-card-matching/">바로가기</a>

자바스크립트 활용 연습용으로 만들어본 카드매칭 게임 입니다.

## 기능

- 플레이어의 이름과 스코어를 Local Storage에 저장합니다.
- 저장된 이름과 스코어를 스코어 순으로 리더보드에 표시해 줍니다
